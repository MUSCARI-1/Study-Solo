# Study-Solo
> 코드 백업 & 공부한 내용들 정리

해킹 워게임 문제 풀이(Write-up)와 알고리즘 문제 해결 코드를 기록하고 관리하는 개인 저장소입니다.

## 전체 디렉토리 구조

```text
Study-Solo/
└── write-up/           # 해킹 워게임 및 보안 스터디 문제 풀이 모음
     ├── dreamhack/      # Dreamhack 문제 풀이
     └── etc/         # 기타 개인 스터디 및 문제
```

## 카테고리별 문제 풀이
현재까지 진행한 워게임 플랫폼별 정리 현황입니다. (폴더 이름을 클릭하면 해당 경로로 이동합니다.)

| 플랫폼 | 구분 | 문제 수 | 비고 |
| :--- | :--- | :---: | :--- |
| [**Dreamhack**](./write-up/dreamhack) | Wargame | 5+ | Level 1~4 진행 중 |
| [**etc**](./write-up/etc) | etc | 3+ | 기타 개인 스터디 및 문제 |
